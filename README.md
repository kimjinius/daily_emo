# daily_emo
-------------------
### 개발 배경
> 지난 한 달 간 자신의 기분이 어땠는지 한번에 볼 수 있는 기분 기록 어플리케이션 개발

### 기능 설명
> 1. ROOM DB를 이용한 정보 저장
> 2. 캘린더를 이용해 날짜 별 일정 저장
> 3. To do List 관리로 해야할 과제 보기 / 진행사항 체크
> 4. 매일매일 기분과 짧은 일기 기록
> 5. 지금까지 저장한 기분을 한 눈에 볼 수 있음
